# Gerenciador-de-tarefas
Aplicação em Java para gerenciamento de tarefas no terminal, permitindo adicionar, visualizar e remover tarefas. Projeto desenvolvido para praticar lógica de programação e estruturas de dados básicas


# Gerenciador de Tarefas em Java

Sistema simples de gerenciamento de tarefas desenvolvido em Java.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas

## Tecnologias utilizadas

- Java
- ArrayList
- Scanner

## Como executar

1. Compile o arquivo:
   java GerenciadorTarefas.java

2. Execute o programa:
   java GerenciadorTarefas
